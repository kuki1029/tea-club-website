
# 🍵 Tea Club Website

The official website for the UW Tea Club!  
This is a modern, mobile-optimized web app built with **React**, **TypeScript**, and **Vite** to help demonstrate and advertise the club's events, services, and ethos.

🌐 **Live Demo:** [tea.imkunal.ca](https://tea.imkunal.ca)

---

## 📱 Features

- ✅ **Responsive Design** – Optimized for mobile devices first
- 📅 **Event Highlights** – Showcase upcoming tea-tasting sessions and social events
- 🫖 **About & Philosophy** – Share the culture and community behind the tea club
- 🧑‍🤝‍🧑 **Join the Club** – Easy way for visitors to express interest or register

---

## 🛠️ Tech Stack

- **React** – Component-based UI
- **TypeScript** – Type-safe development
- **Vite** – Next-gen frontend tooling
- **Tailwind CSS** – Utility-first styling

---

## 🚀 Getting Started

### Prerequisites

- Node.js >= 18.x
- npm 

### Setup Instructions

```bash
# 1. Clone the repository
git clone https://github.com/kuki1029/TeaClubWebsite.git
cd TeaClubWebsite

# 2. Install dependencies
npm install

# 3. Run the development server
npm run dev

# 4. Build for production
npm run build
```

> The dev server will usually start at `http://localhost:5173`.
